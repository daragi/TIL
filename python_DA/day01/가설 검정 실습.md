## 인터랙티브 학습 모듈
### - 에듀테크 플랫폼은 학습자의 몰입도와 학습 성과를 높이기 위해 ‘인터랙티브 학습 모듈’이라는 새로운 기능을 개발했습니다.
 
- 이 기능은 영상 중간에 퀴즈를 삽입하고학습자가 직접 선택·입력하며 진행하도록 설계되었습니다.
- 기능의 효과를 검증하기 위해, 플랫폼은 다음과 같은 실험을 진행했습니다.
 
- 전체 사용자 중 일부를 무작위로 두 그룹으로 나눔
    1. 기존 학습 방식 그룹(A) 인터랙티브 학습 모듈 그룹(B) 두 그룹 모두 동일한 강의 콘텐츠를 수강
    2. 2주간 학습 후 학습 성과 점수를 비교
 
이제 우리는 이 실험 결과를 바탕으로
인터랙티브 `학습 모듈이 실제로 학습 성과를 향상`시켰는지 를
통계적으로 판단하려고 합니다.

----
### 📝 이 실험의목적은 무엇인가요?

answer: 새로 개발한 학습 모듈이 실제로 학습 성과를 향상시켰는지를 판단

### 📝귀무가설과 대립가설을 설정해보세요.

`H0`: 새로 개발한 학습 모듈이 실제로 학습 성과를 향상시키지 않았을 것이다.
`H1`: 새로 개발한 학습 모듈이 실제로 학습 성과를 향상시켰을 것이다.

### 📝이 가설은 추정 문제인가요, 검정 문제인가요?

answer: 성과 향상에 대한 판단이기에 `검정`문제로 봐야한다.

### 📝어떤 검정을 선택하는 것이 적절한가요?

answer: `두 집단의 비교` 이기 때문에 `t-검정`이 유효하다고 생각
answer: 성과의 `증가`를 따져야 하므로 `단측 검정` 유효

### 📝p-value = 0.03일 때, 통계적 판단은?

answer:

1. 유의수준 0.01로 설정한다면 귀무가설을 기각하지 못한다.
2. 유의 수준이 0.05로 설정한다면 귀무가설을 기각함으로 대립가설을 채택할 수 있다.

### 📝 이 결과를 바탕으로 어떤 결정을 내릴 수 있을까요?

answer: 기존의 학습방식 보다 성과가 있었기에 새로운 학습방식을 확대할 수 있음. 그러나 학습방식의 변화에서 오는 비용과 효용의 저울질을 해서 결정해야함

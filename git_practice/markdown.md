## 코드
인라인 코드: 문장 중간에 코드를 삽입하고 싶을 때

>예시
파이썬에서 문장을 출력하고 싶을 때, 사용하는 함수는 `print` 입니다.

블럭 코드: 코드에 언어를 선택하여, syntax highlight를 넣을 수 있다.
```python
for i in range(10):
    print(i)
```

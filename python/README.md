## 📅 커리큘럼 요약

### [Day 01](day01/)
> 파이썬 시작하기
- 데이터 타입과 변수의 개념 이해
- 표준 입출력(`input`, `print`) 활용
- 산술/비교/논리 연산자 및 제어문(`if-else`) 기초

### [Day 02](day02/)
> 데이터의 묶음과 흐름 제어
- 시퀀스 자료형(`List`, `Tuple`, `String`)의 특징과 슬라이싱
- 반복문(`for`, `while`)을 활용한 데이터 순회
- `range()` 함수를 이용한 범위 생성

### [Day 03](day03/)
> 효율적인 데이터 관리와 로직
- 비시퀀스 자료형(`Dict`, `Set`) 및 멤버십 연산자(`in`) 활용
- 다차원 자료구조 탐색 및 데이터 추출
- 기초 알고리즘(최대/최소값 찾기) 및 루프 제어(`break`, `continue`)

### [Day 04](day04/)
> 코드의 재사용과 메서드 활용
- 함수의 정의, 매개변수(`Parameter`)와 반환값(`Return`)
- 익명 함수(`Lambda`)의 활용
- 각 자료형별 필수 메서드(`List`, `String`, `Dict`, `Set`) 학습

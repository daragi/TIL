# 📊 BI Analysis Lab

> 데이터를 **의사결정 가능한 인사이트**로 전환하는  
> BI 분석 및 시각화 실습 아카이빙 공간입니다.

단순 시각화가 아니라  
**문제 정의 → 지표 설계 → 분석 → 인사이트 → 실행 제안**  
까지의 **실무형 분석 흐름**을 기록합니다.

---

# 🎯 Objective

- 비즈니스 문제를 **데이터 질문**으로 구조화
- 핵심 KPI 정의 및 지표 설계
- 시각화 기반 인사이트 도출
- **의사결정에 연결되는 분석 결과 제시**
